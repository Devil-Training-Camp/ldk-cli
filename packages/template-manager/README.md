# @ldk-cli/template-manager

Template manager of `@ldk-cli/cli`.

All options:

```
Usage: ldk-cli temp <action> [nameOrPath]

Manage templates

Options:
  -h, --help  display help for command

<action>:
  --add  Add template
  --remove  Remove template
  --update  Update template
  --show  Show template
```
